# HexRefine Docs

This folder is the project-local documentation boundary for HexRefine.

The project is intended to build, test, and run from this directory without
depending on documentation files from the workspace it was split from. Add new
HexRefine documents here and link them from this README.

Useful entry points:

- `../README.md`: project overview and commands
- `../RELEASE.md`: release packaging and static web hosting
- `architecture.md`: module structure, data ownership, and runtime flow
- `features.md`: supported mesh, refinement, export, GUI, and replay features
- `github-pages.md`: static website deployment with GitHub Pages
- `import-mesh.md`: starting-mesh import formats and constraints
- `implementation.md`: template, planner, merge, conformance, and session details
- `macos-desktop.md`: portable macOS desktop app packaging
- `positioning.md`: comparison with professional preprocessing tools
- `windows-desktop.md`: portable Windows desktop app packaging
- `../src/README.md`: source module map
- `../examples/README.md`: generated examples and browser workbench
